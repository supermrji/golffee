# Elite Golf Inspired Design System

## Visual Theme & Atmosphere
Takes luxury golf and electric automotive as its base, sharpened through subtraction, full-viewport photography, and near-zero UI.
- **Mood**: High-tech, minimalist, spacious, performance-driven.

## Color Palette & Roles
- **Canvas (Background)**: `#ffffff` for light mode, `#000000` for dark mode. High contrast true black/white.
- **Surface**: Subtle overlays (`rgba(255, 255, 255, 0.05)`) and zebra strips (`rgba(255, 255, 255, 0.02)`) for readable data tabular tracks.
- **Text (Foreground)**: `#171a20` on light backgrounds, `#f4f4f4` on dark backgrounds.
- **Accent**: `emerald-400` / `emerald-500` (#34d399 / #10b981) to represent the manicured green of elite golf courses.

## Typography Rules
- **Family**: Universal Sans, Gotham, or system fallbacks.
- **Headings**: Extremely light weights (Font Weight 300).
- **Body**: Wide spacing for microcopy.

## Component Stylings
- **Buttons**:
  - Ghost styles: Transparent background with solid borders. No rounded corners.
- **Forms & Inputs**: Only active borders visible on focus.

## Layout Principles
- **Grid**: Full bleed, edge-to-edge. Negative overlapping margins (e.g. `-mt-32`) allows data panels to float atop breathtaking photography logic without forcing scrolling.
- **Depth**: Soft borders and backdrop filters, zero drop shadow.

## Do's and Don'ts
- **DO**: Use extreme negative space. Color table headers softly, odd/even row contrast is required for data density.
- **DON'T**: Use bright playful colors outside of the core accent. Avoid thick containers or grid lines for table boundaries.
